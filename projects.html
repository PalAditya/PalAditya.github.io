Swayam-Sevak

Smart India Hackathon Project (Cisco), 2019

Have you ever realized how much time we spend in public places asking pretty useless questions?

"Hi, so toilet is to the left?"
"Err, how much time left for the next train?"
"Can you give me the number of Doctor Ashok?"

Swayam Sevak is the ultimate solution to that. Any one can deploy this system locally, and as soon as a person walks into their WI-Fi space (of a hospital, railway station, etc.), they will get connected to a chatbot which would answer all this basic queries instantly. No more bottlenecks due to slow response, or awkwardly trying to approach the help desk either!

StockPlay:Play with shares

A complete web based platform with stock-market emulator features (2019)

Remember that tiny Pset7 from CS50 course for making a finance webapp?

Well, I do, and I liked it a lot. So much so, that I decided to make it even more awesome and a full-fledged platform.

Some features supported by the app are:

URL Function
login Allows user to log in
register User Registration
download Downloads history of transactions in PDF format
index Shows curren shares
quote See current price of a share
buy, sell Buy or sell n particular shares
live See updates of user's most important shares graphically

and many more. It is a complete business solution for a web-app with server and database connections configured
